## 🏡 Pangisha SmartHouse
- A modern React-based house hunting web application that allows users to browse, filter, and save property listings. Built using React, Tailwind CSS, Framer Motion, and SweetAlert2 with data from a public API (DummyJSON).

## Install required packages
- npm install
- npm install react-router-dom
- npm install sweetalert2
- npm install framer-motion
- npm install -D tailwindcss postcss autoprefixer

## API Used
- https://dummyjson.com/products
